Product Image URL Manager

Paste image URLs for each product. Images will be saved and displayed on product detail pages.

📰 Shared data (makikita sa lahat ng browser)

Kung naka-set na ang Firebase (sa firebase-config.js): pag nag-Save ka dito, awtomatikong na-sync sa cloud. Kapag binuksan ang site sa ibang browser o device, makikita rin ang mga images.

Kung wala pang Firebase: gamitin ang Export/Import. I-Export pagkatapos mag-Save, tapos sa ibang browser i-Import ang file.
